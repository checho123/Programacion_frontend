¿Quiénes somos?

Coworking2 se crea a raíz de una necesidad fundamental que hoy en día el mercado Colombiano ignora: El uso profesional de páginas webs dinámicas, optimizadas y adaptables que permitan establecer vínculos más cercanos con los clientes y volver la empresa más competitiva en el mercado laboral.

Misión

Brindar soluciones en diseño, desarrollo, mantenimiento y posicionamiento orgánico de páginas web, ofreciendo un portafolio de servicios de alta calidad y que se adapte a los diferentes presupuestos, caracterizándonos por nuestro cumplimiento y por velar por la completa satisfacción de nuestros clientes.

Visión

Para el año 2021 estar posicionados en el mercado colombiano como la primera opción en creación y administración de sitios web, gracias a la calidad, confianza y credibilidad que ofrecemos a nuestros clientes.

{Tiempo de aprender}

HTML

Por sus siglas en ingles (HiperText Markup Language) en español (Lenguaje de Marcas de Hipertexto) , es el lenguaje sobre el cual está fundamentada la Web, ya que en él se aloja todo el contenido que irá dentro una página y por tanto es el primer lenguaje que se recomienda aprender para entrar al mundo de la programacióny el desarrollo Web.


Ahora bien, dicho contenido debe organizarse de tal modo que pueda ser interpretado correctamente por un navegador, para esto se requiere la utilización de unos elementos llamados “etiquetas”. Las cuales cumplen la funcion de marcar las diferentes partes del contenido para que así el navegador pueda identificar de que tipo se trata (título, párrafo,imagen, video, etc.).

CSS

Por sus siglas en íngles (Cascading Style Sheets) en español ( Hojas de estilo en Cascada), es un lenguaje que describe cómo se va a mostrar la estructura del documento de HTML en la pantalla del usuario como lo es la información, imágenes, tablas, videos, etc. De acuerdo a como se desee modificar el diseño en la hoja de estilos.


Aquí puedes crear animaciones, cambiar tamaños, utilizar el tipo de letra que desees, usar fondos, cambiar colores, usar bordes, alinear texto, márgenes etc. Modificando la apariencia de tu página web de acuerdo a tu necesidad o preferencia de una forma muy sencilla.

Javascript

Es un lenguaje de programación que te permite crear contenido dinámico, controlar archivos multimedia, crear imágenes animadas, cajas de dialogo, validación de formularios y muchas otras cosas.


Se utiliza para dar ese dinamismo o interactividad que genera mayor contacto con el usuario y genere un mayor enganche lo que puede llegar a generar que este permanezca más tiempo en nuestro sitio web.


Se recomienda que para manejar javascript es fundamental saber de antemano HTML y CSS

Frontend

Es la parte del sitio web donde los usuarios pueden interactuar de forma directa.


Se puede ver el diseño, los contenidos y es una de las dos partes en las que se divide la estructura de cualquier Página Web.


Dentro de esta se encuentra todos los lenguajes de programación para Frontend vistos hasta el momento que son HTML, CSS y Javascript, sin contar que para complementar lo anterior, se puede recurrir a los Frameworks y librerías que permiten agilizar el diseño de la página para mejorar la experiencia del usuario.

Frameworks

Es una herramienta de trabajo estandarizada que ayudan a reducir los tiempos de maquetación de una página web, es como armar un rompecabeza, solamente se puede dar algunos retoques o personalizarlo de acuerdo a sus gustos en color de fuentes, tipo de textos y algunas animaciones, ya que vienen prediseñados y adaptarlas a tus requerimientos de acuerdo a lo que vayas a diseñar.


Dentro de los tipos de frameworks que existen, se puede usar Bootstrap, un programa creado por la empresa Twitter, donde tienen diseños de plantillas como tipografías, formas, barras de navegación entreotros aspectos

Librerías

JQuery es un complemento de JavaScript, pero JavaScript lo usa como una librería de código abierto, es usado principalmente para dar efectos y animaciones a lo que estemos trabajando como botones, imágenes, texto, entre otro contenido.


Es rápido, pequeño y una apariencia rica en código, además se actualiza constantemente. Es de código abierto y compatible con diferentes navegadores, es fácil de usar, ahorra tiempo, esfuerzos, integral otros complementos que permite realizar animaciones y efectos personalizados. Además, con la masificación de los smartphones también es usado este código jQuery para dar un mejor aspecto visual de las páginas web en ese tamaño de pantallas.
